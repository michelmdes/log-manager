#Log Manager (Frontend do sistema Log-manager)

## Tecnologias
Spring Boot 2.1.1
Thymeleaf
Bootstrap

## Configurações
Configurar a URL do backend no atributo *urlLogServices* da classe MainController.java 
Essa aplicação está configurada para rodar na porta 8090. Link de acesso:
http://localhost:8090
